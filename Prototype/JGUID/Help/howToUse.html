JGuiD User Guide

JFrame is the default and only option.

You may begin a new project or open an existing project file

Here we have menubar, toolbox, user screen, and properties window.

The JGuiD toolbox provides users with all of the available SWING components

Toolbox allows users to select required components to be added to the user panel by clicking on it.

Components can be moved by dragging with mouse on the user panel.

Components can be resized by dragging at the right-bottom corner of the component.

User can also resize the user panel if it is set Resizable.

The properties window has two tabs. One contains the properties panel of the JFrame, and the other is the properties panel for the currently selected component on the user panel.

The code screen shows the generated java code for the current user screen layout.

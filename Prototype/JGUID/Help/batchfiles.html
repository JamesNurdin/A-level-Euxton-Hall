Batch Files

A batch file is a standard text document saved with the extension .bat

Inside the document, the programmer can put Command Line instructions which will then be executed when the batch file is run. For Java projects, This can be used to store javac and java commands and recompile or run code with a single click.

The command 'pause' can also be used to halt the batch file between instructions.

JGuiD will currently create a batch file for your project automatically when exported, saving it in the same location as the exported .java file
